MacBook Pro 1st generation (2007)

The first generation MacBook Pro used the same design as the Aluminum PowerBook G4, but had an Intel Core 2 Duo instead of a PowerPC G4.

  • Intel Core 2 Duo 2.2 GHz processor
  • 2 GB RAM
  • 120 GB hard drive
  • DVD-RW SuperDrive
  • 15" TFT LCD screen
  • Weight: 2.4 kg
